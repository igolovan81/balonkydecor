# Wishlist Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let visitors bookmark products in a session-backed wishlist (no login), view them at `/{lang}/wishlist`, and add non-variant products to the cart from there.

**Architecture:** A new static `Wishlist` service (mirrors the existing `Cart` service) stores a flat array of SKUs in `$_SESSION['wishlist']`. A new `WishlistController` renders the wishlist page and handles a POST-redirect-GET toggle endpoint. Heart-icon toggle forms are added to the shop grid and product detail page, driven by a `wishlist_skus` array passed from `ShopController`.

**Tech Stack:** Slim 4, Twig 3, PHP 8, PDO/MySQL, PHPUnit 11, vanilla CSS/JS (no build step).

## Global Constraints

- Wishlist is session-only — no DB table, no customer accounts (spec: Constraints).
- Wishlist stores SKUs only, never a specific subtype (spec: Constraints).
- All interactions are plain HTML `<form method="POST">` + full page reload — no AJAX/JS (spec: Constraints).
- New public routes go under `/{lang}/...`; admin routes are untouched (`.claude/rules/backend.md`).
- Every visible string goes through `t('key')`, added to **all five** `lang/{cs,en,ru,uk,sk}.json` files with identical key sets (`.claude/rules/frontend.md`).
- All public links are language-prefixed (`.claude/rules/frontend.md`).
- Session-specific pages get `<meta name="robots" content="noindex,nofollow">` (existing convention: cart, checkout, order status).
- Model/service tests run against real Docker MySQL, no mocks; unique/`INSERT IGNORE` fixtures (`.claude/rules/unit-testing.md`).
- Interactive elements are real `<a>`/`<button>`, never clickable `<div>`/`<span>`; toggle buttons need `aria-label` (`.claude/rules/frontend.md`).
- CSS: tokens from `:root` for repeated colors, flat kebab-case classes, `--modifier` for variants, `.active`/`.is-open` for JS/toggle states (`.claude/rules/css-styling.md`).

```php
<?php
namespace App\Services;

use App\Models\ProductModel;

class Wishlist
{
    private static function boot(): void
    {
        if (session_status() === PHP_SESSION_NONE) {
            session_start();
        }
        if (!isset($_SESSION['wishlist'])) {
            $_SESSION['wishlist'] = [];
        }
    }

    public static function toggle(string $sku): bool
    {
        self::boot();
        $key = array_search($sku, $_SESSION['wishlist'], true);
        if ($key !== false) {
            unset($_SESSION['wishlist'][$key]);
            $_SESSION['wishlist'] = array_values($_SESSION['wishlist']);
            return false;
        }
        $_SESSION['wishlist'][] = $sku;
        return true;
    }

    public static function has(string $sku): bool
    {
        self::boot();
        return in_array($sku, $_SESSION['wishlist'], true);
    }

    public static function skus(): array
    {
        self::boot();
        return $_SESSION['wishlist'];
    }

    public static function count(): int
    {
        self::boot();
        return count($_SESSION['wishlist']);
    }

    public static function items(string $lang): array
    {
        self::boot();
        $items = [];
        foreach ($_SESSION['wishlist'] as $sku) {
            $product = ProductModel::findBySku($sku, $lang);
            if ($product !== null) {
                $items[] = $product;
            }
        }
        return $items;
    }
}
```

```php
<?php
namespace App\Controllers;

use App\Services\Wishlist;
use Psr\Http\Message\ResponseInterface as Response;
use Psr\Http\Message\ServerRequestInterface as Request;

class WishlistController extends BaseController
{
    public function index(Request $request, Response $response, array $args): Response
    {
        $lang = $request->getAttribute('lang');

        return $this->render($request, $response, 'public/wishlist.twig', [
            'items' => Wishlist::items($lang),
        ]);
    }

    public function toggle(Request $request, Response $response, array $args): Response
    {
        $lang   = $request->getAttribute('lang');
        $body   = (array) $request->getParsedBody();
        $sku    = trim($body['sku'] ?? '');
        $return = (string) ($body['return'] ?? '');

        if ($sku !== '') {
            Wishlist::toggle($sku);
        }

        $target = preg_match('#^/[a-z]{2}/#', $return) ? $return : "/{$lang}/wishlist";

        return $response->withHeader('Location', $target)->withStatus(302);
    }
}
```
